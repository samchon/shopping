## Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS 4
- **UI Components**: Custom components using Tailwind
- **API Client**: @samchon/shopping-api (SDK generated by Nestia)

## Getting Started

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Create `.env.local`** with API configuration:
   ```
   NEXT_PUBLIC_API_HOST=http://127.0.0.1:37001
   NEXT_PUBLIC_APP_NAME="Shopping Mall"
   NEXT_PUBLIC_APP_URL=http://localhost:3000
   ```

3. **Run the development server**:
   ```bash
   pnpm dev
   ```

4. **Open your browser** at http://localhost:3000

## Key Features

- **Shared Layout**: Consistent header, navigation, and footer across all pages
- **Product Catalog**: Browse products with search functionality
- **Shopping Cart**: View and manage cart items
- **Authentication**: Login and signup pages for customer management

## Adapters & Abstraction

The frontend follows a clean architecture pattern:

1. **API Adapter Layer** (`src/lib/api.ts`): Wraps the raw SDK to provide a clean interface
2. **Domain Models**: Normalized types for business objects
3. **UI Components**: Reusable, composable components
4. **Pages**: Route-specific implementations that use adapters

## Development Commands

- `pnpm dev` - Start development server
- `pnpm build` - Build production bundle
- `pnpm start` - Start production server
